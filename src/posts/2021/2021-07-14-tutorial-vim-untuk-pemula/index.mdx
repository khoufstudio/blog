Vim (**Vi IMproved**) adalah teks editor hasil kloning dari **Vi** teks editor disertai dengan beberapa tambahan. 
Vim dibuat oleh Bram Moolenaar dan pertama kali rilis pada tahun 1991. 
Vim merupakan aplikasi _open source_ yang tersedia untuk banyak sistem operasi, diantaranya seperti Linux, Windows dan OSX.  

Vim merupakan teks editor yang dijalankan di terminal/cli/cmd. Ketika menggunakan Vim, seluruh aktifitas dilakukan dengan menggunakan keyboard 
(walaupun, sebenarnya bisa saja mengaktifkan fitur mouse). Meskipun demikian, dengan menggunakan Vim kita bisa menambah dan merubah teks dengan lebih efisien.

## Mode di VIM
Vim memiliki tiga mode: **Normal**, **Visual**, dan **Insert** 

**Normal Mode**  
Normal mode merupakan mode pertama saat pertama kali membuka file dengan Vim. Di normal mode kita tidak dapat menambahkan karakter apapun. 
Mode ini juga disebuat mode perintah (_command mode_) karena apa yang kita ketik akan dianggap sebagai suatu perintah. 
Misal, ketika kita mengetik 'k' maka tidak akan menampilkan huruf 'k', melainkan cursor akan bergerak ke atas. 

Untuk beralih ke Normal mode, kita dapat menekan tombol **Esc**.

**Insert Mode**  
Insert mode merupakan mode untuk menambahkan/mengedit/menghapus teks di dalam suatu file. 
Setiap karakter yang telah diketik akan muncul di tempat kursor berada. 
Untuk beralih ke mode ini, ketika sedang berada di normal mode adalah dengan mengetik tombol **i**.

**Visual Mode**  
Visual mode adalah mode yang memiliki fungsi untuk memblok teks dan melakukan berbagai operasi. 
Misal _cut_, _copy_, dan _delete_. Untuk beralih ke visual mode, kita dapat menekan tombol **v** pada _command mode_.
