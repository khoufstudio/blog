Pada tutorial kali ini saya ingin berbagi tentang cara membuat zip di ubuntu via cli. Berikut langkah-langkahnya:

1. Install zip

`````
$ sudo apt-get install zip
`````

2. Gunakan perintah ini untuk membuat zip: `zip -r {filename.zip} {foldername}`. Contoh: 
`````
$ zip -r folder_to_zip.zip folder_to_zip/
`````

Sekian tutorial singkat ini. Semoga bermanfaat!
